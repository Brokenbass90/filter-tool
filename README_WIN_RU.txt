Vadim Filter Tool — запуск на Windows (коротко)
==============================================

Требуется:
1) Google Chrome или Microsoft Edge
2) Node.js LTS (если не установлен — установится автоматически)

ПЕРВЫЙ ЗАПУСК
1. Распакуйте архив, например в: C:\vadim-tool
2. Запустите файл START_WIN.bat (двойной клик)
3. Если Node.js не установлен — откроется установщик, нажимайте "Next" -> "Install"
4. После установки запустите START_WIN.bat ещё раз
5. Подождите установку зависимостей (1–2 минуты)
6. Откроется браузер с программой
7. Окно "Strong Bridge OCR Server" оставьте открытым (оно нужно для OCR)

ПОВТОРНЫЙ ЗАПУСК
— Просто двойной клик по START_WIN.bat

ВАЖНО
• Если OCR не работает, проверьте, что окно "Strong Bridge OCR Server" не закрыто.
• START_WIN.bat сам очищает зависший процесс/порт 17871 и заново запускает helper.
• START_WIN.bat проверяет версию Node.js: при v24+ попросит поставить LTS 20/22.
• При смене версии Node.js скрипт сам переустанавливает helper-зависимости.
• Внутри программы статус должен быть "OCR: подключен".
• Данные хранятся в браузере (Chrome/Edge). Не очищайте кэш/данные сайта, иначе база пропадет.

ЕСЛИ NODE.JS НЕ УСТАНОВЛЕН
— Скрипт сам запустит установщик (nodejs-lts-x64.msi).
— После установки запустите START_WIN.bat ещё раз.

ЕСЛИ НУЖНО ОСТАНОВИТЬ OCR
— Запустите STOP_WIN.bat (закроет OCR сервер).

EMAIL РАССЫЛКА ИЗ ПРОГРАММЫ (ЛИЧНЫЙ GMAIL)
1) В Gmail включите 2FA (двухэтапная аутентификация).
2) Создайте App Password (16 символов) для "Mail".
3) В программе откройте вкладку "Сообщение":
   - Gmail логин (example@gmail.com)
   - App Password
   - Тема письма (можно с плейсхолдерами)
   - Лимит в день (рекомендуется 200-300)
4) Выберите карточки на вкладке "Карточки".
5) На вкладке "Сообщение" нажмите "Проверить лимит Email".
6) Выберите режим кнопкой:
   - "Просто сообщение" (ручные сообщения),
   - "HTML письмо" (email-шаблон).
7) Нажмите кнопку отправки.

ПРИМЕЧАНИЯ
• Без собственного домена доставляемость ниже, часть писем может попадать в спам.
• Программа контролирует дневной лимит и показывает остаток.
• Для быстрого теста есть файлы: test-data\test-email-card-1.html и test-data\test-email-card-2.html


КАК НАСТРОИТЬ GMAIL ДЛЯ ОТПРАВКИ (ПРОСТО)
1) Входите в тот же Gmail, который будет в поле логина в программе.
2) Включите 2FA в аккаунте Google.
3) Создайте App Password: https://myaccount.google.com/apppasswords
   - Приложение: Mail
   - Устройство: Other -> Vadim Tool
4) В программу вставляйте:
   - Логин: тот же Gmail
   - Пароль: только этот App Password (16 символов, без пробелов)
5) Сначала отправьте 1 тестовое письмо.

ТИПОВЫЕ ОШИБКИ EMAIL
• 535 BadCredentials
  Логин и App Password от разных аккаунтов, или пароль введен неверно.
• 534 Application-specific password required
  Введен обычный пароль Gmail, нужен именно App Password.
• Важно: пара всегда должна совпадать:
  - user@gmail.com + App Password user@gmail.com

ШАБЛОНЫ БЕЗ "\n"
• В текст сообщения вставляйте обычные переносы Enter.
• Если вставили текст с символами "\n", программа сама преобразует их в нормальные переносы строк при сохранении/отправке.
